Projects

Secure remote garage door controller implemented on raspberry pi
My final project for CSC 5250 - Network, Distributed, and Concurrent Programming
In this project a garage door which simulated by a raspberry pi can be controlled by an app on your phone using a secured SSL connection.
Files for this project can be download here.

Dynamic network design and reconfiguration for Wireless Sensor Networks using different topologies
My final project for CSC 7290 - Advanced Computer Networking
In this project I have simulated a WSN which dynamically reconfigures to change some of the configuration of each node. This method can also be used to dynamically generate a network based on different topologies like star.
Full report of this project can be found here.

Interaction between ARQ and coding: retransmission versus heavier coding
My final project for CSC 6290 - Data Communication and Computer Networks
There is a trade off between hard coding or retransmission of messages in a computer network. In this project I have studied the impact of focusing on each of these methods.
Full report of this project can be found here, also the code for this project available here.

Event Scheduler
This website is designed to share schedules, arrange a meeting or sign-up for volunteer events. Features like edit, delete, invite and tracking invitations is available. This project was done for The Medical Education Support Group at 2014 and can be used by Wayne State students and staff.
